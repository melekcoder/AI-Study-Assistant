# AI Study Assistant

An AI-powered study assistant application that helps students organize their learning process.

## Project Goal

This project aims to help students:

- Manage their courses
- Save their notes
- Plan their tasks
- Generate summaries with AI
- Create questions with AI

## Technologies

- Python
- Flask
- HTML / CSS
- MySQL
- Git & GitHub
